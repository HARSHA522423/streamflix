# StreamFlix 🎬

StreamFlix is a Netflix-inspired movie streaming web application built using HTML, CSS, JavaScript, Spring Boot, and MySQL.

## Features
- Netflix-style UI
- Auto sliding banner
- Trending movies section
- Dynamic movie cards
- Play button overlay
- Search and profile icons
- Multi-language movie collection
- Responsive dark theme UI

## Technologies Used
- HTML5
- CSS3
- JavaScript
- Spring Boot
- MySQL
- REST API

## Future Improvements
- User authentication
- Watchlist feature
- Movie search API
- Video streaming support
- Mobile responsive design
